# agents-dev-env

A portable development environment based on Ubuntu 24.04 LTS, providing comprehensive support for multiple programming languages and development tools.

## Languages
- Python with uv package manager
- Node.js with npm, pnpm, yarn, and bun
- Go
- Rust
- Java (OpenJDK)
- PHP
- Ruby
- Perl

## Build Tools
- GCC and Clang compilers
- CMake and Ninja build systems
- Make

## Development Tools
Frequently updated tools managed via mise:
- Claude Code
- OpenCode
- Codex
- GitHub CLI

## Usage
Build the Docker image and run containers for consistent development environments across different machines.
